# vBulletinJDownloaderForumHelper
Helper script (Userscript) for faster moderation of JDownloader support forums

## Features
* Quickly alter thread prefixes: 1 click or hotkey vs 6 clicks
* Quickly alter thread title
* Quickly apply changes with hotkey **CTRL + E**
* Quickly copy thread id with button or hotkey **CTRL + SHIFT + C**

## Installation
1. Get Tampermonkey: 
2. Click here to install script: https://raw.githubusercontent.com/pspzockerscene/vBulletinJDownloaderForumHelper/refs/heads/main/vBulletinJDownloaderForumHelper.user.js
3. Visit threads in JDownloader forums while logged in as moderator to use the script.
